## Library Management
 
 # AddBooks.jsx
      Add books by submitting the form details. 
 # BookSlice.jsx
     created bookslice with addBook action
 # Browsebooks.jsx
     browse books using category 
 # Header.jsx
    includes nav bar and outlet
 # HomePage.jsx
    welcome message, links to books category,popular books cards
 # Navigation.jsx
    Nav bar with home,browse books, add books links
 # NoFound.jsx
    Error msg if wrong url
 # store.jsx
     Configured store and added add book reducer
 # ViewDetails.jsx
     Getting id from the url and displaying the matched book by title or author
 # data.js
     Dummy data for library
 # App.jsx
     Routes to the pages


 # To run the project
   install react,react-router-dom,tailwind,react-redux,redux toolkit
   npm run dev
